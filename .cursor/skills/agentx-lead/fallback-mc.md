# Fallback multiple choice (when AskQuestion is unavailable)

Use this only if the AskQuestion tool cannot be called. Still keep **Other** last.

```markdown
**Q1 — <plain question>?** *(recommended: B)*

A) …
B) … — recommended
C) …          *(optional third)*
Other) I’ll type what I want

Reply with e.g. `Q1B` or `Q1 Other: …`
```

For extras:

```markdown
**Q2 — Which should we include?** *(pick any)*

1) Max failed attempts + lockout — recommended
2) Password reset — recommended
3) Email verification
Other) …

Reply with e.g. `Q2: 1,2` or `Q2 Other: …`
```
