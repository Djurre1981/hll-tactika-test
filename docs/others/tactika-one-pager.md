# Tactika - All-in-one Competition hub for The Circle

**What it is:** Tactika is The Circle’s private Hell Let Loose strategy and prep platform. Steam sign-in, member-only access — our climbs, strats, routes, and match plans stay inside the team.

**Why it exists:** Prep used to live in Discord threads, PDFs, and personal drives. Tactika puts it on the maps we already know, so everyone can find it and use it before match night.

**Where:**

[hll-tactika.pages.dev](https://hll-tactika.pages.dev) (Interactive map tool)
[hll-tactika-test.pages.dev](https://hll-tactika-test.pages.dev) (test hub)

---

## What you can use today

| Tool               | What it does                                                                           |
| ------------------ | -------------------------------------------------------------------------------------- |
| **Hub**            | Home dashboard — upcoming events, your open prep tasks, online presence, tool launcher |
| **Calendar**       | Scrims, comps, practices — opponent, map, faction, result                              |
| **Match Brief**    | One page per match: facts, notes, linked tools, prep checklist                         |
| **Climbing Guide** | Climb & MG spots on all 20 maps, with video                                            |
| **Stratmaker**     | Multi-slide tactical drawings (routes, callouts, HLL placeables) — live co-edit        |
| **Micro Prep**     | Whiteboards & slideshows for freeform match prep — live co-edit                        |
| **Routeplanner**   | Timed truck routes with pathfinding, frontier wall, playback                           |
| **Team / Roster**  | Comp rosters (drag-and-drop), site access, staff management                            |
| **HLL Records**    | Past match history — filters, win/loss, **My matches**, open any Brief                 |

**Match night flow (shipped):** Calendar or Hub → **Match Brief** → attach strats / routes / whiteboards / roster → assign prep tasks → members tick them off under **My tasks**.
