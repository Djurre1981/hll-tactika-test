# Climbing Guide

**Goal:** find (or publish) climb and MG spot pins with video on HLL maps.

## Open the guide

From the hub, open **Climbing Guide** (`/climbing-guide-v1/`). This is the original map module, still used alongside V2 tools.

## Browse pins

1. Pick a map.
2. Filter by faction / pin type as needed.
3. Hover for preview; click for full video and details.

Pins cover **climbs** and **MG spots**, with multi-media where available.

## Edit pins (Advisor+)

1. Enter **editor mode** (role-gated).
2. Place, drag, or update pins; use undo/redo.
3. Comp Assist+ can edit others’ / seed pins; Advisors edit their own.

Members can always view; they cannot enter editor mode.
