# Micro Prep

**Goal:** freeform whiteboard or slideshow prep for briefings (map backgrounds optional).

## Modes

| Mode | Canvas | Typical use |
|------|--------|-------------|
| **Whiteboard** | Square | Free sketch / brainstorm |
| **Slideshow** | 16:9 multi-slide | Briefing decks |

Open from the hub → **Micro Prep**.

## Drawing

Core draw tools match Stratmaker (pen, shapes, text, etc.) without HLL placeable icons / ping.

You can:

- Compose an **HLL map background** (map + overlays → stored page)
- Use image backgrounds and dark/light theme
- Collaborate live on objects (Yjs)

## Link to a match

Link a calendar event from the side panel, or attach the whiteboard from **Match Brief**.

Respect **locks** on the event or the whiteboard when edits are blocked.
