# Records & HeLO

**Goal:** browse match history, filter by team / “my matches,” and understand HeLO imports.

## Records

Hub → **Records** for the match history panel.

- Filter by team (**Circle** / **Circle Jr**)
- **My matches** — games where your Steam64 appears on our side
- Links toward HeLO / CRCON when stored on the event

Combat stats (K/D form, etc.) feed Management player views when imported.

## HeLO import (staff)

Historical Circle and Circle Jr series can be imported into the calendar with participant Steam IDs and combat stats. Operational detail for maintainers: repo doc `docs/helo-import.md`.

Members don’t run imports — if a match is missing, ask Admin/Owner.

## Strat history

Management → **History** → **Strat history** lists strats with search/sort by date, opponent, map, faction, strongpoint.
