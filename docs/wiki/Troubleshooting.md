# Troubleshooting

Quick fixes for common blockers.

## I signed in but see forbidden / blank access

You are not on the Steam allowlist (or your role was removed). Ask an **Owner** to add you in **Admin Panel → Team**.

## I can’t create or edit a strat / event / pin

- **Comp Member** is read-only for content. You need **Comp Advisor** or higher.
- The event or tool may be **locked** — ask Assist/Admin/Owner to unlock, or open a copy if that’s the workflow.
- Advisors can only edit **their own** strats/pins; Assists can edit any.

## I don’t see Management

Management is **Admin / Owner** only. Use hub tools and Match Brief instead, or ask staff for a roster/history answer.

## Raincheck / RSVP button does nothing

Refresh once; if it still fails, check you’re logged in and the event isn’t in a broken state. Report to Admin with the event name/time.

## “You played” / My matches is empty

Your Steam ID must appear on **our side** of imported HeLO/history stats. Spreadsheet roster alone does not mark you as played. Ask Admin if history import is incomplete.

## Routeplanner won’t let me set a start

Select a **route** first — start/HQ controls are per-route, not plan-level.

## Obstacle mode blocks route edits

That’s expected. Exit obstacle mode to edit paths again.

## Help / wiki won’t open

Use the **?** control immediately left of your avatar. If overlays are blocked by another modal, close that modal first (Esc).

## Still stuck

Ping staff on Discord with: page URL, what you clicked, and your Steam name.
