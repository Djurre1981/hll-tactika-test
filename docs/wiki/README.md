# Wiki source (`docs/wiki`)

Canonical member manual for the **in-app help** overlay (avatar **?**).

## Conventions

- Flat page names with hyphens (`Calendar-and-Match-Brief.md`).
- `_Sidebar.md` drives in-app nav.
- Screenshots use markdown images that render as **dark placeholders** in-app (do not commit real screenshot files).
- Prefer task-based how-tos over dumping every feature flag.
