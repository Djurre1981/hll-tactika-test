# Management

**Goal:** staff ops — roster, history, analytics, and folders. **Comp Admin** and **Owner** only.

Open **Management** from the hub (hidden for Members / Advisors / Assists).

## Sections

| Section | Use it for |
|---------|------------|
| **Overview** | Season pulse, attendance / player-form ranked lists, period filters (30d / 1y / All) |
| **Roster** | Shared Comp Roster (Circle + Circle Jr), status, authorization (site roles), import/seed |
| **Folders** | Organize staff content |
| **History** | **Match history** and **Strat history** (search/sort) |
| **Analytics** | Win/loss charts by month, map, opponent |

## Roster tips

- Import from Sheets or seed from HeLO when bulk-updating the pool.
- **Authorization** (Owners): set viewer / editor / assist / admin for site access — does not invent Steam accounts; allowlist still required.
- Spreadsheet / HeLO roster rows are **not** automatic site grants.

## Overview period

Use the shared period control so Attendance, Player form, and Season pulse stay aligned. “All” uses loaded history (typically up to ~24 months of match history in the client).
